## [1.0.0] - 2026-02-13
### Added
- Initial release of the SWIFT SaaS Theme.
- Fully responsive Astro + Tailwind layout.
- Global mobile navigation with public JS script.
- Production-ready Pricing component with demo content.
- Contact page with form UI and toast notifications.
- Header, Footer, and modular component structure.
- Global CSS and consistent design system.
- README with installation, customization, and deployment instructions.
- MIT License included.

### Fixed
- Mobile menu spacing and stacking behavior.
- Script loading issues by moving JS to `/public`.
- Footer spacing and layout alignment.

### Notes
- Pricing section is demo-only; no payment processing or backend included.
- Toast component uses React but requires no configuration.
